# Spotify_Web_App
Projekt Spotify Web App realizowany w ramach pracy inżynierskiej.

SpotiTools to nowoczesna, stylizowana w klimacie cyberpunkowym aplikacja muzyczna, łącząca minimalistyczny design z neonową estetyką. Projekt stawia na wizualną czytelność, prostotę i estetykę. Obsługuje tryby jasny i ciemny, animowane tła, przejścia i efektowne mikrointerakcje.

🔧 Główne cechy:

    🎨 Motywy jasny/ciemny z neonową kolorystyką

    🎧 Interaktywne sekcje, animowane SVG, efekty hover

    🧩 Skalowalny i przejrzysty kod CSS z wykorzystaniem zmiennych

    🖱️ Intuicyjny interfejs oparty o nowoczesny layout i komponenty

    🌌 Styl inspirowany estetyką cyberpunk / synthwave
