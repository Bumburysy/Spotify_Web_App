# SpotiTools – Frontend

## Opis projektu

SpotiTools to aplikacja webowa, której celem jest rozszerzenie możliwości zarządzania muzyką na platformie Spotify. Projekt powstał jako część pracy inżynierskiej i obecnie prezentuje koncepcję, stylistykę oraz wstępny zakres funkcjonalności aplikacji. W kolejnych etapach planowana jest rozbudowa o kolejne moduły oraz integracja z backendem i Spotify API.

## Aktualny stan

### Obecna wersja zawiera:

- Nowoczesny, responsywny interfejs z obsługą motywów (ciemny/jasny)
- Animacje i efekty wizualne (neon, pulsacje, cząsteczki)
- Podstawową strukturę nawigacji (sidebar, topbar)
- Przykładowe sekcje tematyczne
- Przygotowane miejsce pod integrację z backendem i Spotify API
  
### Struktura plików
- index.html – główny plik HTML z sekcjami tematycznymi i strukturą strony
- styles.css – rozbudowany plik stylów z obsługą motywów, animacjami i responsywnością
- scripts.js – skrypt obsługujący interaktywność, przełączanie motywów, animacje, sidebar, wyszukiwarkę itp.
- logo.png – logo projektu
- audio/ – przykładowe pliki muzyczne do demonstracji (opcjonalnie)

### Technologie

- HTML5, CSS3 (zmienne CSS, animacje, media queries)
- JavaScript (ES6+)
- Przygotowanie pod integrację z React/Vue/Angular oraz backendem (Node.js/Python/Java)

### Jak uruchomić

- Sklonuj repozytorium: https://github.com/Bumburysy/Spotify_Web_App/
- Otwórz plik index.html w przeglądarce.

### Plany rozwoju

- Integracja z Spotify Web API (OAuth2)
- Moduł analityczny i rekomendacji
- Zaawansowane zarządzanie playlistami
- Backend (Node.js/Express, Python lub Java)
- Wizualizacje danych (D3.js/Chart.js)
- Wersja mobilna

### Dokumentacja

- Wszystkie kluczowe zmiany i kolejne etapy rozwoju będą dokumentowane w repozytorium na platformie GitHub.

## Uwaga:

- Projekt jest w fazie rozwojowej – obecna wersja prezentuje głównie warstwę frontendową i koncepcję graficzną. - Dalsze funkcjonalności będą sukcesywnie dodawane.
